# Personal Project Showcase Platform

A modern, responsive Single Page Application (SPA) built with React. This platform allows creative agencies or individuals to showcase their work and dynamically update their portfolio as new projects are completed.

## Features
*   **Project Dashboard:** A landing page that displays a list of all current projects.
*   **Dynamic Project Entry:** A form-based system that allows users to add new projects to the showcase in real-time without reloading the page.
*   **Real-time Search:** A search feature that dynamically filters the project list based on user input.
*   **Responsive Design:** A layout inspired by the provided design mock-ups that works seamlessly on desktop and mobile devices.

## Tools and Technologies
*   **Frontend:** React.js
*   **Styling:** CSS3 (Flexbox/Grid)
*   **Version Control:** Git & GitHub

## Setup and Usage Instructions
1.  **Clone the Repository:**
    ```bash
    git clone <your-github-repo-link>
    ```
2.  **Install Dependencies:**
    Navigate to the project folder and run:
    ```bash
    npm install
    ```
3.  **Start the Application:**
    Run the following command to launch the app in your browser:
    ```bash
    npm start
    ```
    *(Or `npm run dev` if using Vite)*

## Known Limitations
*   Data is currently stored in local state; refreshing the page will reset the project list to the default state.
*   Image uploads are represented by placeholders in this version.
